# vue2-shop-hero

> A Vue2 project by HEROWALKING

### 技术栈：
1. vue2
2. vue-router
3. vue-lazyload
4. vue-infinite-scroll@
5. axios
6. vuex
7. express
8. MongoDB

---

### Build Setup

#### 数据库
1. 启动本机MongoDB
2. 通过mongorestore恢复数据库数据

#### Server:
1. cd server
2. npm install
3. npm start

#### Client:
1. npm install
2. npm start
